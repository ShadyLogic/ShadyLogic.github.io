downto

Iterates the given block, passing decreasing values from int down to and including limit. If no block is given, an Enumerator is returned instead.


even?

Returns true if int is an even number.


floor

As int is already an Integer, all these methods simply return the receiver. Synonyms are to_int, floor, ceil, truncate.


gcd

Returns the greatest common divisor (always positive). 0.gcd(x) and x.gcd(0) return abs(x).


gcdlcm

Returns an array; [int.gcd(int2), int.lcm(int2)].


integer?

Since int is already an Integer, this always returns true.


lcm

Returns the least common multiple (always positive). 0.lcm(x) and x.lcm(0) return zero.


next

Returns the Integer equal to int + 1, same as #next.


numerator

Returns self.


odd?

Returns true if int is an odd number.


ord

Returns the int itself. This method is intended for compatibility to character constant in Ruby 1.9. For example, ?a.ord returns 97 both in 1.8 and 1.9.


pred

Returns the Integer equal to int - 1.


rationalize

Returns the value as a rational. The optional argument eps is always ignored.


round

Rounds int to a given precision in decimal digits (default 0 digits). Precision may be negative. Returns a floating point number when ndigits is positive, self for zero, and round down for negative.


succ

Returns the Integer equal to int + 1, same as #next.


times

Iterates the given block int times, passing in values from zero to int - 1. If no block is given, an Enumerator is returned instead.

to_r

truncate

upto
